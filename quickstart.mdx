## Setup your development

Follow these steps to get started with your documentation:

### 1. Clone the Repository

During onboarding, we created a GitHub repository containing your docs content. Find it on our [dashboard](https://dashboard.mintlify.com) and clone it locally.

### 2. Preview Changes

1. Install the Mintlify CLI:
```bash
npm i -g mintlify
```

2. Run the preview server:
```bash
mintlify dev
```

### 3. Deploy Changes

1. Install our GitHub app from the [dashboard](https://dashboard.mintlify.com) to enable automatic deployments
2. Push your changes to GitHub to update your docs site
