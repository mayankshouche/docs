## Image

### Utilisation de Markdown

La [syntaxe markdown](https://www.markdownguide.org/basic-syntax/#images) vous permet d'ajouter des images en utilisant le code suivant

```md
![titre](/chemin/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5MB. Sinon, nous recommandons d'héberger sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pouvez ensuite utiliser cette URL et l'intégrer.

### Utilisation des Intégrations

Pour obtenir plus de personnalisation avec les images, vous pouvez également utiliser les [intégrations](/writing-content/embed) pour ajouter des images

```html
<img height="200" src="/chemin/image.jpg" />
```

### iFrames

Charge une autre page HTML dans le document. Le plus souvent utilisé pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
